<h1> Вебзастосування "Список завдань" </h1>


<p> Вебзастосування "Список завдань" дає змогу вести облік виконання  завдань, оперуючи ними на одній сторінці. У вебзастосуванні реалізовано можливість добавляти нові завдання та видаляти непотрібні, збереження напрацьованого списку завдань при оновленні сторінки, після закриття вкладки або перезавантаження комп'ютера за рахунок його збереження у LocalStorage. Вебзастосування створено за допомогою НТМL, CSS та JavaScript.</p>

## 🛠️ Локальний запуск

<br>
<p> Для перегляду вебзастосування локально послідовно виконайте наступні команди:</p>
<ol>
  <li> <code> git clone  https://github.com/tarasporovhuck/app-task-list-javascript.git </code>, щоб клонувати репозиторій на локальний комп’ютер; </li>
  <li> <code> cd app-task-list-javascript </code>, щоб перейти в папку app-task-list-javascript; </li>
  <li> <code> start index.html </code>, щоб переглянути вебзастосування у браузері. </li>
</ol>



## 📦 Розгортання (CI/CD)


Проєкт використовує **GitHub Actions** для автоматичного розгортання оновленої версії вебзастосування на GitHub Pages.
Кожен `push` у гілку `main` ініціює робочий процес, який оновлює версію застосунку на GitHub Pages.
Конфігурація знаходиться у файлі: `.github/workflows/deploy.yml`.
