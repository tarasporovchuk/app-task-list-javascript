# 📝 Вебзастосування "Список завдань"


Простий та інтуїтивно зрозумілий менеджер завдань, який працює безпосередньо у вашому браузері. У вебзастосуванні реалізовано можливість добавляти нові завдання та видаляти непотрібні. Цей проєкт створений для демонстрації навичок роботи зі статичними вебтехнологіями.


## 🌐 Демо-версія


Переглянути вебзастосування можна за посиланням:
👉 **https://tarasporovchuk.github.io/app-task-list-javascript/**

## 🛠️ Локальний запуск


Для локального перегляду проекту необхідно виконати наступні кроки:


1.  **Клонувати репозиторій app-task-list-javascript.git:**
    ```bash
    git clone https://github.com/tarasporovhuck/app-task-list-javascript.git
    ```


2.  **Перейти у папку app-task-list-javascript:**
    ```bash
    cd app-task-list-javascript
    ```


3.  **Відкрити файл index.html у браузері:**
    ```bash
    start index.html    
    ```


## 📦 Розгортання (CI/CD)


Проєкт використовує **GitHub Actions** для автоматичного розгортання оновленої версії вебзастосування на GitHub Pages.
Кожен `push` у гілку `main` ініціює робочий процес, який оновлює версію застосунку на GitHub Pages.
Конфігурація знаходиться у файлі: `.github/workflows/deploy.yml`.
